# RAG_finance
# دستیار هوش مالی (Financial Knowledge AI Assistant)

**نام فایل نوت‌بوک:**  
`financial_rag_assistant.ipynb`

---

## توضیح پروژه

این پروژه با هدف ساخت یک دستیار هوش مصنوعی برای پاسخ‌گویی به سوالات در حوزه **سواد مالی و هوش مالی** طراحی شده است.  
این سیستم از منابع معتبری مانند **مقالات یک وب‌سایت و کتاب‌های مالی** به عنوان پایگاه دانش استفاده می‌کند.

کاربران می‌توانند سوالات خود را درباره مفاهیم مالی بپرسند و سیستم پاسخ‌هایی مبتنی بر منابع موجود ارائه خواهد داد.

هسته اصلی این پروژه بر پایه معماری **RAG (Retrieval Augmented Generation)** طراحی شده است.

در این معماری:

1. محتوای مقالات و کتاب‌ها جمع‌آوری و پردازش می‌شود.
2. متن‌ها پاکسازی شده و به بخش‌های کوچک‌تر (**Chunk**) تقسیم می‌شوند.
3. برای هر بخش **Embedding** تولید می‌شود.
4. این Embeddingها در یک **Vector Database** ذخیره می‌شوند تا جستجوی معنایی امکان‌پذیر شود.

زمانی که کاربر سوالی می‌پرسد:

1. سوال به یک embedding تبدیل می‌شود.
2. مرتبط‌ترین بخش‌های متن از پایگاه داده برداری بازیابی می‌شوند.
3. این اطلاعات به یک **مدل زبانی بزرگ (LLM)** داده می‌شود.
4. مدل بر اساس اطلاعات بازیابی‌شده پاسخ نهایی را تولید می‌کند.

این معماری کمک می‌کند پاسخ‌ها بر اساس منابع واقعی تولید شوند و احتمال تولید پاسخ‌های نادرست (Hallucination) کاهش یابد.

## ویژگی‌های اصلی سیستم

- استخراج و پردازش داده از مقالات وب‌سایت
- استخراج متن از کتاب‌های مالی
- پاکسازی و آماده‌سازی داده‌ها
- تقسیم متن به بخش‌های کوچک (Chunking)
- تولید Embedding برای جستجوی معنایی
- ذخیره داده‌ها در Vector Database
- جستجوی معنایی در میان منابع
- تولید پاسخ با استفاده از مدل زبانی
- امکان نمایش منبع پاسخ
- سیستم پرسش و پاسخ در حوزه آموزش مالی

## کاربرد سیستم

این سیستم می‌تواند به عنوان یک **دستیار آموزشی هوشمند** در یک وب‌سایت یا پلتفرم آموزشی مالی استفاده شود.

کاربران می‌توانند درباره موضوعاتی مانند موارد زیر سوال بپرسند:

- سواد مالی
- مدیریت پول
- سرمایه‌گذاری
- استقلال مالی
- برنامه‌ریزی مالی شخصی

سیستم پاسخ‌ها را بر اساس دانش استخراج‌شده از مقالات و کتاب‌ها ارائه خواهد داد.

## هدف نهایی پروژه

هدف نهایی این پروژه ساخت یک **پایگاه دانش مالی هوشمند** است که بتواند دانش موجود در مقالات و کتاب‌های مالی را به یک **دستیار تعاملی مبتنی بر هوش مصنوعی** تبدیل کند.

این سیستم به کاربران کمک می‌کند تا سریع‌تر، ساده‌تر و تعاملی‌تر به دانش مالی دسترسی پیدا کنند.
